# OTP Input Component 🔐

A sleek and user-friendly OTP (One-Time Password) input component for web applications. This component provides a smooth experience for entering verification codes with automatic focus management and validation. ✨

## Features 🚀
- **Auto-focus navigation**: Automatically moves to the next input field as you type
- **Input validation**: Ensures only numeric values are entered
- **Responsive design**: Works seamlessly on desktop and mobile devices
- **Customizable styling**: Easy to integrate with your existing UI
